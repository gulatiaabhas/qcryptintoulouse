# Welcome to QCrypt in Toulouse

This website provides information and updates for the Quantum Cryptography Workgroup in Toulouse. 

TIQTOQS, the Quantum@Toulouse will start to organise talks on Quantum Cryptography. We will mostly be following the TU Delft Open Course by Stephanie Werner and Thomas Vidick, ![Here](https://ocw.tudelft.nl/courses/quantum-cryptography/)

Stay tuned for more details and news!
